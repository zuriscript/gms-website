# Introduction to GMS

> Learn about GMS, how to install it and how to use it.
> GMS is meant to be a research, evaluation and testing platform and encourages best practices.
> Feel free to tweak things your way...

GMS is the first benchmarking suite for graph mining that facilitates evaluating and constructing high-performance graph mining algorithms.
It comes with a benchmark specification and offers a modular approach to graph representations, set representations, preprocessing routines and algorithmic subroutines.
